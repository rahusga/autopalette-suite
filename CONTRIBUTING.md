# Contributing

Thanks for your interest in AutoPalette Studio.

For now, the preferred way to contribute is by opening a GitHub Issue with:

- PixInsight version
- Operating system
- AutoPalette Studio version
- Input data type: OSC dual-band, Ha/OIII, Ha/OIII/SII
- Whether the image was linear or non-linear
- Steps to reproduce the problem
- Screenshot or console log if available

Pull requests are welcome, but please open an issue first for major changes.
